예제 문제

  1. ( 하이퍼링크 )는 인터넷에서 문서 사이를 쉽게 이동할 수 있는 기능입니다.
  2. ( W3C )는 HTML 표준을 비롯한 웹 표준안을 제작하거나 제안하는 일을 하는 국제적인 웹 표준화 단체입니다.
  3. ( 웹브라우저 )는 웹에 공개된 정보를 탐색할 수 있게 하는 프로그램입니다.
  4. ( 플러그인 )은 사용자 PC에 프로그램을 추가로 설치해 웹 브라우저 기능을 확장하는 방법입 니다.
  5. ( 웹2.0 )은 서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대를 의미합니다.
  6. ( WHATWG )는 Web Application 1.0 표준을 작성한 단체입니다. 현재는 HTML W/G라는 이름으로 변경했습니다.
  7. Web Application 1.0 표준은 새로운 웹 표준으로 채택된 후 ( HTML5 ) 표준이라는 이름 으로 변경되었습니다.
  8. 웹은 ( 요청 )과 ( 응답 )이라는 간단한 형태로 동작합니다.
  9. ( URL )은 웹에서 어떤 대상을 구분하는 주소를 의미합니다.
  10. ( URI )는 웹에서 어떤 대상을 구분하는 방법들을 총칭합니다.
  11. 요청하는 쪽을 ( 클라이언트 )(사용자)라고 하며, 응답하는 쪽을 ( 서버 )(제공자)라고 합니다.
  12. 서버 프로그램은 사용자와 먼 곳에 있기 때문에 ( 백엔드 ) 프로그램이라고 합니다.
  13. 클라이언트 프로그램은 사용자와 가까운 곳에 있기 때문에 ( 프론트엔드 ) 프로그램이라고 합니다.
  14. ( 마크업 )은 웹 페이지의 서식이나 구조를 표현하는 정보입니다.
  15. ( CSS )는 HTML 페이지에 스타일을 지정하는 스타일시트를 작성할 때 사용하는 언어 입니다.
  16. ( ECMAScript )는 자바스크립트의 표준 명칭입니다.
  17. ( 자바스크립트 )는 HTML 페이지에서 사용자 반응 등을 처리하는 스크립트를 작성하는 언어입 니다.
  18. ( 일렉트론 )은 HTML5 기반의 데스크톱 애플리케이션 개발 엔진입니다.
  19. ( 리액트 네이티브 )는 페이스북에서 만든 모바일 애플리케이션을 개발할 수 있게 하는 엔진입니다.
  20. ( 크로스 플랫폼 )은 한 번 개발해서 여러 장치(플랫폼)에 대응하는 것을 의미합니다.

  2. 모든 URI는 URL입니다. ( X )
  3. 2000년대 초에는 액티브X와 플래시 덕분에 웹 브라우저로 다양한 것을 할 수 있었습니다. ( O )
  4. 자바스크립트를 사용하면 웹 페이지에 프로그래밍 요소를 부여해서 사용자와 반응을 할 수 있습니다. ( O )
  5. 통신을 더욱더 강화하기 위해 온라인 상태에서만 사용할 수 있게 되었습니다. ( X )
  6. 그레이디언트, 애니메이션 등 화려한 스타일을 CSS3로 추가할 수 있게 되었습니다. ( O )
  7. 단방향 통신뿐만 아니라 양방향 통신도 가능해졌습니다. ( O )
  8. 배터리 잔량 같은 장치 정보도 가져올 수 있게 되었습니다. ( O )
  9. 웹 브라우저에서 작동하는 클라이언트 프로그램은 기본적으로 C++, 자바, C#, 파이썬 등 프로그램으로 개발합니다. ( X )
